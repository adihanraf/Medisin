API Testing dengan Postman

Untuk melakukan pengujian API, gunakan Postman Collection yang sudah disediakan.

File Postman Collection: https://github.com/adihanraf/Medisin/blob/main/BE-Laravel/Medisin.postman_collection.json

Cara menggunakan:

Unduh atau buka file Postman Collection dari link di atas.

Buka Postman → Import → Pilih file JSON.

Semua endpoint API siap untuk dites di Postman.
